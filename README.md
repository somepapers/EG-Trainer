# EG-Trainer — 二阶EG法公式训练工具

针对 2x2 魔方 EG (Erik-Gunnar) 解法的专项公式训练器，支持 **EG-0 (CLL)** 、 **EG-1** 、 **EG-2** 三套公式体系，共 114 条公式。内置 **WCA 计时器**（支持观察时间、罚时、ao5/ao12 统计）和 **预判练习器**（打乱预判正确率统计）。

## 在线使用

👉 **[somepapers.github.io/EG-Trainer](https://somepapers.github.io/EG-Trainer)**

## 本地使用

直接用浏览器打开 `index.html` 即可使用，无需安装任何依赖或启动服务器。

## EG 方法简介

EG 法是 2x2 魔方最主流的一步解法：先做好一面（Face），然后根据底层角块的排列情况分为三种子集：

| 子集 | 底层状态 | 公式数 | 说明 |
|------|---------|--------|------|
| **EG-0** | 底层已完全还原 | 40 条 | 即 CLL，顶层 7 种 OLL × 最多 6 种PLL |
| **EG-1** | 底层有一组邻角交换 | 40 条 | 7 种 OLL × 最多 6 种排列 |
| **EG-2** | 底层有一组对角交换 | 34 条 | 5 组（S/AS/Pi/U/L），U组含10个case |

## 使用方式

### 公式训练

1. 打开 `index.html`，选择 EG-0 / EG-1 / EG-2 进入训练
2. 勾选需要练习的 case 类型（H / Pi / T / L / U / S / AS）
3. 工具显示打乱公式（即目标算法的逆），执行该算法即可还原
4. 点击 **做对了** 自动 2.5 秒后切换下一条；点击 **做错了** 显示标准公式
5. 切换到 **统计** 标签查看各 case 正确率和历史记录
6.建议使用PC端打开网站食用，手机端部分功能未做适配

### 计时器

首页点击 **⏱️ 计时器** 进入。生成 WCA 标准的 9 步二阶打乱公式，用于测速练习。

- **按住空格键**准备，**松开空格键**开始计时
- 再次**按下空格键**停止计时
- 停止后可选择罚时：**无罚时 / +2 / DNF**
- 可开启 **15 秒观察时间**（超时自动 +2，超过 18 秒判 DNF）
- 统计页显示 **最佳成绩**、**ao5**、**ao12** 及完整成绩列表

### 预判练习器

首页点击 **🧠 预判练习器** 进入。生成 WCA 标准打乱，用于练习观察预判能力。

- 查看打乱公式后进行预判
- 点击 **做对了 / 做错了** 记录结果
- 底部显示正确数、错误数和正确率统计

### 键盘快捷键

**公式训练页：**

| 快捷键 | 功能 |
|--------|------|
| `→` / `J` | 做对了 |
| `←` / `K` | 做错了 |
| `空格` | 下一条打乱 |
| `C` | 复制打乱公式 |

**计时器：**

| 快捷键 | 功能 |
|--------|------|
| `空格`（按下） | 准备 / 停止计时 / 开始下一次 |
| `空格`（松开） | 开始计时 |
| `←` / `→` | 停止后切换罚时选项 |
| `Enter` | 确认罚时 |

**预判练习器：**

| 快捷键 | 功能 |
|--------|------|
| `→` / `J` | 做对了 |
| `←` / `K` | 做错了 |
| `空格` | 下一条打乱 |

## 数据说明

- 所有公式数据存储在 `script.js` 的 `DATA` 对象中，按 `EG0` / `EG1` 分别组织
- 每个 case 的 `show_correct` 为该 case 的①号公式（做错时展示），`_cases` 包含全部备选公式（打乱时随机抽取）
- 公式来源于 2011CAOS01 整理的公式库
- 如需替换公式，直接编辑 `script.js` 中 `DATA` 对象对应位置即可

## 本地存储

- 统计记录保存在浏览器 localStorage
- EG-0 使用 key `eg0_case`，EG-1 使用 key `eg1_case`，EG-2 使用 key `eg2_case`
- 计时器成绩使用 key `eg_timer_records`（最多保留 500 条）
- 预判练习器统计使用 key `eg_preinspect_stats`
- 各模块清空统计互不影响

## 数据导入导出

支持将训练成绩导出为 JSON 文件备份，或从之前导出的文件恢复数据。

1. 进入 **统计** 页面
2. 点击 **📥 导出成绩** 将当前所有统计数据保存为 `.json` 文件
3. 点击 **📤 选择导入文件** 选取之前导出的 JSON 文件即可恢复数据（导入将覆盖当前数据）

导出的文件包含：EG-0 / EG-1 / EG-2 各 case 的正确/错误计数、计时器成绩记录、预判练习器统计等全部历史数据。

## 程序更新注意事项

程序更新可能导致界面异常或统计数据显示错误（通常是 localStorage 数据结构不兼容所致）。如遇此类问题，请按以下步骤恢复：

1. **更新前**：先点击 **📥 导出成绩**，将当前所有数据导出为 JSON 文件备份
2. **更新后**：若出现界面问题，进入统计页面点击各模块的 **清空统计** 按钮，清除异常数据
3. 点击 **📤 选择导入文件**，导入步骤 1 中导出的备份文件即可恢复全部记录

> ⚠️ 建议定期导出成绩备份，避免浏览器缓存清理或程序更新导致数据丢失。

## 打乱规则

打乱公式 = 算法公式的**逆**（反转顺序并反转每个转动方向）：
- `R` → `R'`，`R'` → `R`，`R2` → `R2`
- 例如 `R U R'` 的打乱为 `R U' R'`
