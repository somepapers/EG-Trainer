# EG-Trainer — 二阶EG法公式训练工具

针对 2x2 魔方 EG (Erik-Gunnar) 解法的专项公式训练器，支持 **EG-0 (CLL)** 、 **EG-1** 、 **EG-2** 三套公式体系，共 114 条公式。

## 在线使用

👉 **[somepapers.github.io/EG-Trainer](https://somepapers.github.io/EG-Trainer)**

## 本地使用

直接用浏览器打开 `index.html` 即可使用，无需安装任何依赖或启动服务器。

## EG 方法简介

EG 法是 2x2 魔方最主流的一步解法：先做好一面（Face），然后根据底层角块的排列情况分为三种子集：

| 子集 | 底层状态 | 公式数 | 说明 |
|------|---------|--------|------|
| **EG-0** | 底层已完全还原 | 40 条 | 即 CLL，顶层 7 种 OLL × 最多 6 种PLL |
| **EG-1** | 底层有一组邻角交换 | 40 条 | 7 种 OLL × 最多 6 种排列 |
| **EG-2** | 底层有一组对角交换 | 34 条 | 5 组（S/AS/Pi/U/L），U组含10个case |

## 使用方式

1. 打开 `index.html`，选择 EG-0 / EG-1 / EG-2 进入训练
2. 勾选需要练习的 case 类型（H / Pi / T / L / U / S / AS）
3. 工具显示打乱公式（即目标算法的逆），执行该算法即可还原
4. 点击 **做对了** 自动 2.5 秒后切换下一条；点击 **做错了** 显示标准公式
5. 切换到 **统计** 标签查看各 case 正确率和历史记录

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `→` / `J` | 做对了 |
| `←` / `K` | 做错了 |
| `空格` | 下一条打乱 |
| `C` | 复制打乱公式 |

## 数据说明

- 所有公式数据存储在 `script.js` 的 `DATA` 对象中，按 `EG0` / `EG1` 分别组织
- 每个 case 的 `show_correct` 为该 case 的①号公式（做错时展示），`_cases` 包含全部备选公式（打乱时随机抽取）
- 公式来源于 2011CAOS01 整理的公式库
- 如需替换公式，直接编辑 `script.js` 中 `DATA` 对象对应位置即可

## 本地存储

- 统计记录保存在浏览器 localStorage
- EG-0 使用 key `eg0_case`，EG-1 使用 key `eg1_case`
- 清空统计仅清除当前模式的数据，不影响另一个模式

## 打乱规则

打乱公式 = 算法公式的**逆**（反转顺序并反转每个转动方向）：
- `R` → `R'`，`R'` → `R`，`R2` → `R2`
- 例如 `R U R'` 的打乱为 `R U' R'`
